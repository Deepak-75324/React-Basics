# React Learning Journey 🚀

This repository contains the projects I built while learning React.js. The goal of these projects is to understand React fundamentals, hooks, state management, component-based architecture, and how React applications work.

## 📚 What I Learned

- JSX and React Components
- Props and State
- Event Handling
- Conditional Rendering
- Lists and Keys
- React Hooks
  - useState
  - useEffect
  - useRef
  - useCallback
- Custom Hooks
- API Integration
- Form Handling
- Component Reusability
- Project Structure and Best Practices

---

## 🛠️ Projects

### 1. Counter App 🔢
A simple counter application to learn state management using `useState`.

#### Features
- Increment Count
- Decrement Count
- Reset Counter

#### Concepts Learned
- React Components
- useState Hook
- Event Handling

---

### 2. Background Color Changer 🎨
A project that changes the background color with button clicks.

#### Features
- Multiple Color Options
- Dynamic UI Updates

#### Concepts Learned
- State Management
- Inline Styling
- Event Handling

---

### 3. Currency Converter 💱
A currency conversion application using API data.

#### Features
- Convert Between Currencies
- Real-Time Exchange Rates
- User-Friendly Interface

#### Concepts Learned
- API Fetching
- useEffect Hook
- Form Handling
- State Management

---

### 4. Password Generator 🔐
A secure password generator with customizable options.

#### Features
- Adjustable Password Length
- Include Numbers
- Include Special Characters
- Copy Password to Clipboard

#### Concepts Learned
- useState
- useCallback
- useRef
- Clipboard API

---

### 5. Custom React Hooks ⚛️
Projects created to understand and implement custom hooks.

#### Examples
- useCurrencyInfo
- useLocalStorage
- useFetch

#### Concepts Learned
- Reusable Logic
- Custom Hook Creation
- Hook Best Practices
